GetDvdχ2ℓ

Method (Function, i.e. spin, matrix, time evolution, etc.) Parent Class etc.

This method calculates and returns \(  \partial v \over \partial \chi_{2 \ell } \) given \(v\) and \( \partial \dot{v} \over \partial \chi_{2 \ell } \).

Technical description:
This method calculates and returns \( \partial v \over \partial \chi_{2 \ell } \) given \(v\) and \( \partial \dot{v} \over \partial \chi_{2 \ell } \). See the theoretical description for how a formula for \( \partial v \over \partial \chi_{2 \ell } \) was derived.
Parameter Type Description
v Double Our post newtonian expansion factor \(v = (M \omega)^\frac{1}{3} \).
dvdχ2ℓdot Double \( \partial \dot{v} \over \partial \chi_{2 \ell } \), the time derivative of \( \partial v \over \partial \chi_{2 \ell } \).
Return Type Description
dvdχ2ℓ Double \( \partial v \over \partial \chi_{2 \ell } \)
Theoretical description:
The formula for \( \partial v \over \partial \chi_{2 \ell } \) was obtained by rearranging Equation (8) of the DerivativeExpressions document to solve for \( \partial v \over \partial \chi_{2 \ell } \). 
Usage locations:
This method is called only once in the InitDvdχ2ℓ method of the Evolver class to intialize \( \partial v \over \partial \chi_{2 \ell } \). For all other calculations of \( \partial v \over \partial \chi_{2 \ell } \) the leapfrog method is used. 
Usage example:
GetDvdχ2ℓ(vN, dvdχ2ℓdot0)
