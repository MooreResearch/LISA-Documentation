MethodSpin CaseEquationSolver

This method calculates and returns \( \partial v \over \partial \delta \) given \(v\) and \( \partial \dot{v} \over \partial \delta \).

Technical description:
This method calculates and returns \( \partial v \over \partial \delta \) given \(v\) and \( \partial \dot{v} \over \partial \delta \). See the theoretical description for how this is done. 
Parameter Type Description
v Double Our post newtonian expansion factor \(v = (M \omega)^\frac{1}{3} \).

dvdδdot

 Double \( \partial \dot{v} \over \partial \delta \)
Return Type Description
dvdδ Double \( \partial v \over \partial \delta \)
Theoretical description:
The derivation of the equation used to calculate \( \partial v \over \partial \delta \) given \(v\) and \( \partial \dot{v} \over \partial \delta \) can be found by rearranging Equation (1) in the DerivativeExpressions document for \( \partial v \over \partial \delta \).
Usage locations:
This method is used in the InitDvdδ method of the Evolver class to initialize the first two values of \( \partial v \over \partial \delta \). For all of the rest of the values of \( \partial v \over \partial \delta \) the leapfrog method is used. 
Usage example:
GetDvdδ(vN,dvdδdotN)
