EquationSolver Constructor

Method Spin Case EquationSolver

Initializes all relevant properties of the EquationSolver Class

Technical description:
Initializes c1 - c25, δ, η, χs, χa, dηdδ, dχsdδ, dχsdχ1ℓ, dχsdχ2ℓ, dχadδ, dχadχ1ℓ, dχadχ2ℓ, χ1 and χ2. Descriptions of these values can be found on the EquationSolver class page.
Parameter Type Description
None N/A N/A
Return Type Description
None N/A N/A
Usage locations:
This method is called each time a new instance of the EquationSolver class is created. Most importantly an instance of this class is created in the constructor of the Evolver class, which is used throughout that method to aid in calculations.
