GetDvdδdot

Method Spin Case EquationSolver

This method calculates and returns \( \partial \dot{v} \over \partial \delta \).

Technical description:
The purpose of this code is to determine \( \partial \dot{v} \over \partial \delta \): the rate of change of \( \partial v \over \partial \delta \) with respect to a dimensionless time \( \tau_s = \frac{t_{s}}{M} \) in the source frame.  See the theororetical description for information about how a formula for \( \partial \dot{v} \over \partial \delta \) was derived.
Parameter Type Description
v Double Our post newtonian expansion factor \(v = (M \omega)^\frac{1}{3} \).

dvdδ

 Double \( \partial v \over \partial \delta \)
Return Type Description
dvdδdot Double \( \partial \dot{v} \over \partial \delta \)
Theoretical description:
The derivation for how \( \partial \dot{v} \over \partial \delta \) can be calculated can be found in Equation (1) of the DerivativeExpressions document.
Usage locations:
This method is used in the InitDvdδ method of the Evolver class to initialize the first two values of \( \partial v \over \partial \delta \), it is also used in the DoStep method to propegate \( \partial v \over \partial \delta \) through time. 
Usage example:
GetDvdδdot(vN,dvdδN)
