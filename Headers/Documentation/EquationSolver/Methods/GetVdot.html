GetVdot

MethodSpin CaseEquationSolver

This method calculates and returns \(\dot{v}\) given \(v\).

Technical description:
This method calculates and returns \(\dot{v}\), the derivative of \(v\) with respect to a dimensionless time \( \tau_s = \frac{t_{s}}{M} \) in the source frame, given \(v\).
Parameter Type Description
v Double Our post newtonian expansion factor \(v = (M \omega)^\frac{1}{3} \)
Return Type Description
vdot Double The derivative of \(v\) with respect to a dimensionless time \( \tau_s = \frac{t_{s}}{M} \) in the source frame.
Theoretical description:
See Equation (1.9) of the Theory1 document for an expression for \(\dot{v}\).
Usage locations:
This method is used in the InitV, InitDvdδ, InitDvdχ1ℓ, and InitDvdχ2ℓ methods to intialize the first two values of each parameter. It is also used in the DoStep method to propegate \( v \) through time.
Usage example:
GetVdot(vN)
