GetΩ2Coef

MethodSpin CaseEquationSolver

This method calculates the scalar that \( \hat{L}_N \) must be multiplied by to obtain \( \vec{\Omega}_2\).

Technical description:
This method calculates the scalar that \( \hat{L}_N \) must be multiplied by to obtain \( \vec{\Omega}_2 \). This method cleans up calculations done in the Evolver class's DoStep method.
Parameter Type Description

dτ

 Double A small step of dimensionless time \( \tau_s = \frac{t_{s}}{M} \) in the source frame.
Return Type Description
Ω2Coef Double The coefficient that will multiply  \( \hat{L}_N \) to obtain \( \vec{\Omega}_2 \).
Theoretical description:
See Equation (1.29) of the Theory1 document to see the expression for \(\vec{\Omega}_2 \), as well as how it is used to calculate the rate of change of star 2's angular momentum.
Usage locations:
This method is called in the Evolver class's InitSpinVariables method to intialiaze \( \hat{L}_N \). It is also called in the DoStep method to propegate \(\hat{ \chi_2 } \).
Usage example:
GetΩ2Coef(dτ)
