GetLNhatdot

Method Spin Case EquationSolver

This method calculates and returns \(\dot{\hat{L}}_N \)

Technical description:
The purpose of this code is to determine \(\dot{\hat{L}}_N \): the rate of change of \(\hat{L}_N\) with respect to a dimensionless time \( \tau_s = \frac{t_{s}}{M} \) in the source frame. The code first defines two vectors:  \( \vec{v}_1 = |\vec{\chi_1}|(1+\delta)^2 \vec{\Omega}_1 \times\hat{\chi}_1 \) and  \( \vec{v}_2 = |\vec{\chi_2}|(1-\delta)^2 \vec{\Omega}_2 \times \hat{\chi}_2 \). Then it calculates \( \dot{\hat{L}}_N = - \frac{v}{4 \eta} (\vec{v}_1 + \vec{v}_2) \). Finally the code returns \(\dot{\hat{L}}_N\). See the theororetical description for an expanation on how these equations were derived.
Parameter Type Description
v Double Our post newtonian expansion factor \(v = (M \omega)^\frac{1}{3} \).

χ1hat

 Vector The unit vector that points in the direction of star 1's spin angular momentum.

χ1hat

 Vector The unit vector that points in the direction of star 2's spin angular momentum.

Ω1

 Vector \( \vec{\Omega}_1 \) is a quantity useful in calculating the rate of change of star 1's angular momentum. See Equation 1.29b for the definition of \( \vec{\Omega}_1 \).

Ω2

 Vector \( \vec{\Omega}_2 \) is a quantity useful in calculating the rate of star 2's angular momentum. See Equation 1.29b for the definition of \( \vec{\Omega}_2 \).
Return Type Description
LNhatdot Vector The rate of change of the direction of the total angualar momentum of our system. 
Theoretical description:
The derivation for how \( \dot{\hat{L}}_N \) can be calculated can be found on page 9 of the theory doument in Equation 1.31.
Usage locations:
This method is called in the InitSpinVariables method of the evolver class, and in the DoStep method of the evolver class.
Usage example:
GetdLNhatdot(vN, χ1hatN, χ2hatN, Ω1hatN, Ω2hatN)
