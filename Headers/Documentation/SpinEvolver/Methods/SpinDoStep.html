SpinDoStep

Method (Function, i.e. spin, matrix, time evolution, etc.) Parent Class etc.

This method propegates \(\vec{\chi_1}\), \(\vec{\chi_2}\), and \(\hat{L}_N\), and then uses  \(\hat{L}_N\) to calculate the values of \(\iota\) and \(\alpha\). 

Technical description:
This method propegates \(\vec{\chi_1}\) and \(\vec{\chi_2}\) by first calculating \(\vec{\Omega}_1 \) and \(\vec{\Omega}_2 \) using this class's GetΩ1Coef, and GetΩ2Coef functions respectively. The method then uses \(\vec{\Omega}_1 \) and \(\vec{\Omega}_2 \) in Equation (1.29a) to find \(\frac{d\chi_{1}}{d\tau} \) and \(\frac{d\chi_{2}}{d\tau} \), these values are then used in the leapfrog equation to propegate \(\vec{\chi_1}\) and \(\vec{\chi_2}\). The method then propegates \(\hat{L}_N\) using the leapfrog equation and the GetLNhatdot method. \(\iota\) and \(\alpha\) are then re-calculated using this new value of \(\hat{L}_N\), and Equation (1.32) in the Theory1 document. Finally, past and current values are updated.


Parameter Type Description

dτ

 Double A small step in unitless time \(\tau = \frac{t_s}{M} \) where \(t_s\) is time in the source frame.
vN Double The current ("Now") value of the post newtonian parameter \(v = (M\omega)^{1/3} \)
Return Type Description
None N/A N/A
Theoretical description:
See Equation 1.15 for more information on how and why the leapfrog method is used. See equations (1.29a) and (1.29b) for more information on how \(\vec{\Omega}_1 \), \(\vec{\Omega}_2 \), \(\frac{d\chi_{1}}{d\tau} \) and \(\frac{d\chi_{2}}{d\tau} \) are calculated.
