InitDvdδ

Method Spin Case Evolver

This method initializes \( \partial v \over \partial \delta \) by calculating and setting its first two values.

Technical description:
In order to initialize \( \partial v \over \partial \delta \) we must calculate both the past and current value of \( \partial v \over \partial \delta \). The code first calculates the past value of \( \partial \dot{v} \over \partial \delta \) which is the derivative of \( \partial v \over \partial \delta \) with respect to unitless time \(\tau_s = \frac{t_s}{M}\) in the source frame. It does this by creating two paralell simulations with differing values of \( \delta \) ; one simulation uses a value of \( \delta + \Delta \delta \) and the other uses \( \delta - \Delta \delta \). Currently the code uses a value of \( \Delta \delta = 10^{-6}\). The code then performs the cenetered difference between these two paralell simulations to obtain \( \partial \dot{v} \over \partial \delta \). The code then feeds this value to the EquationSolver class's GetDvdδ method which contains an analytic solution for \( \partial v \over \partial \delta \) given \(v_0\) and \( \partial \dot{v} \over \partial \delta \). This becomes the past value of \( \partial v \over \partial \delta \), and Euler's method is used to obtain the present value of \( \partial v \over \partial \delta \).

Theoretical description:
This method provides the initial two values for \( \partial v \over \partial \delta \) that will be used to propegate the simulation forward in time. This method uses the centered difference equation to obtain an initial value for \( \partial \dot{v} \over \partial \delta \), see Equation (1.16) of the Theory1 document for more information on the centered difference method. This method then uses Euler's equation to obtain the second value of \( \partial v \over \partial \delta \). See the discussion surrounding Equation (1.13) in the Theory1 document for more regarding Euler's method.
Usage locations:
This method is called only once in the constructor for the Evolver class.
