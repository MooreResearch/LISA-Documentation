InitV

Method Spin Case Evolver

this method initializes \(v\) by calculating and setting its first two values.

Technical description:
The parameter \(v_0\) is passed into the program by the user and is set as the past value of \(v\). This method then uses the EquationSolver class's GetVdot method to analytically calculate \(\dot{v}\); the derivative of \(v\) with respect to unitless time \(\tau_s = \frac{t_s}{M} \) in the source frame. Using this time derivative and Euler's method, this method then calculates and sets the present value of \(v\).

Theoretical description:
This method is passed \(v_0 \) and uses Euler's method to initialize the second value of \(v\). For all future values of \(v\) the leapfrog method is used. See the discussion surround Equation (1.13) in the Theory1 document for more.
Usage locations:
This method is called once in the constructor of the Evolver class.
