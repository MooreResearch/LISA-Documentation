MethodSpin CaseEvolver

This method initializes \( \partial v \over \partial \chi_{1 \ell} \) by calculating and setting its first two values.

Technical description:
In order to initialize \( \partial v \over \partial \chi_{1 \ell} \) we must calculate both the past and current value of \( \partial v \over \partial \chi_{1 \ell} \). The code first calculates the past value of \( \partial \dot{v} \over \partial \chi_{1 \ell} \) which is the derivative of \( \partial v \over \partial \chi_{1 \ell} \) with respect to unitless time \(\tau_s = \frac{t_s}{M}\) in the source frame. It does this by creating two paralell simulations with differing values of \( \chi_{1 \ell} \) ; one simulation uses a value of \( \chi_{1 \ell} + \Delta \chi_{1 \ell} \) and the other uses \( \chi_{1 \ell} - \Delta \chi_{1 \ell} \). Currently the code uses a value of \( \Delta \chi_{1 \ell} = 10^{-6}\). The code then performs the cenetered difference between these two paralell simulations to obtain \( \partial \dot{v} \over \partial \chi_{1 \ell} \). The code then feeds this value to the EquationSolver class's GetDvdχ1ℓ method which contains an analytic solution for \( \partial v \over \partial \chi_{1 \ell} \) given \(v\) and \( \partial \dot{v} \over \partial \chi_{1 \ell} \). This becomes the past value of \( \partial v \over \partial \chi_{1 \ell} \), and Euler's method is used to obtain the present value of \( \partial v \over \partial \chi_{1 \ell} \).

Theoretical description:
This method provides the initial two values for \( \partial v \over \partial \chi_{1 \ell} \) that will be used to propegate the simulation forward in time. This method uses the centered difference equation to obtain an initial value for \( \partial \dot{v} \over \partial \chi_{1 \ell} \), see Equation (1.16) of the Theory1 document for more information on the centered difference method. This method then uses Euler's equation to obtain the second value of \( \partial v \over \partial \chi_{1 \ell} \). See the discussion surrounding Equation (1.13) in the Theory1 document for more regarding Euler's method.
Usage locations:
This method is called only once in the constructor for the Evolver class.
