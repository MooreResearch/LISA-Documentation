Evolver

Class Time evolution etc.

The Evolver class is meant to evolve four equations to obtain values of \(v\), \(\partial v \over \partial δ\), \(\partial v \over \partial χ_{1ℓ}\), and \(\partial v \over \partial χ_{2ℓ}\) over time. It takes some parameters from the main program and uses the leapfrog method to evolve these equations. Each step of the leapfrog method is then stored as a property that can be accessed by the main program.

Technical Description:
The class's constructor takes the initial timestep, \(v\), \(δ\), \(χ_{1ℓ}\), and \(χ_{2ℓ}\) as parameters. It then uses these values to initialize the four evolutions, and it sets up a new instance of the EquationSolver class that has these base parameters. The initializing methods set up the first two values of each evolution and set those to the be "current" past and present values (which are properties of the class), respectively. Then, any time the DoStep method is called, the Evolver class performs a step using the leapfrog method and updates these past and present values.
