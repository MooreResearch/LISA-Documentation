DoStep

MethodSpin CaseEvolveretc.

Calculates values of v, ∂v/∂δ, ∂v/∂χ1ℓ, and ∂v/∂χ2ℓ dτ in the future by using past and present values in the centered difference equation.

Technical description:
This code uses past and present values of v, ∂v/∂δ, ∂v/∂χ1ℓ, and ∂v/∂χ2ℓ to calculate their future values using the centered difference equation. The time derivatives of v, ∂v/∂δ, ∂v/∂χ1ℓ, and ∂v/∂χ2ℓ used in the centered difference equation are found using the EquationSolver class.  This method then sets the past value to the present value and the present value to the future value. 
Parameter Type Description
dτ Double The time step used in the centered difference equation to propegate v, ∂v/∂δ, ∂v/∂χ1ℓ, and ∂v/∂χ1ℓ
Return Type Description
None N/A N/A
Theoretical description:
This method calculates v, ∂v/∂δ, ∂v/∂χ1ℓ, and ∂v/∂χ2ℓ at a time step dτ in the future by using the centered difference equation:
 \[x_{future} =  x_{past} + 2\Delta \tau \dot{x}_{present}\]
The values of these propegated derivatives will later be used in the Fischer matrix in order to determine the uncertainty in each value.
