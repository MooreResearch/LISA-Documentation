ιαEvolver Constructor

Method (Function, i.e. spin, matrix, time evolution, etc.) Parent Class etc.

This method initailizes the first two values of \(\iota\), \(\alpha\), \(\hat{L}_N\), \( \vec{\chi}_1 \), and \( \vec{\chi}_2 \).

Technical description:
First this method sets the values of \(\delta\) and \(\eta\). Then the intial values of \(\iota\), \(\alpha\), and \(\hat{L}_N\) are computed using Equations (1.32a,b) and Equation (1.7) respectively. Then \(\hat{\chi}_1\) and \(\hat{\chi}_2\) are defined and used the GetLNhatdot method to obtain \(\dot{\hat{L}}_N\) which is used in Euler's method to obtain the "Now" value of \(\hat{L}_N\). Equations (1.32a,b) are then used again to obtain the "Now" values for \(\iota\) and \(\alpha\) from (\hat{L}_N\). Finally, \( d \vec{\chi}_1 \over d \tau \) and \( d \vec{\chi}_2 \over d \tau \) are computed and used in Euler's method to find the "Now" values of \(\vec{\chi}_1\) and \(\vec{\chi}_2\). 
Parameter Type Description

v0

 Double The initial value of the post newtionian parameter \(v = (M\omega)^{1/3} \)

dτ0

 Double The initial value of a small step in unitless time \(\tau = \frac{t_s}{M} \) where \(t_s\) is time in the source frame.

myδ

 Double \( \delta = \frac{ m_1 - m_2 }{m_1 + m_2 } \) a unitless measure of the mass difference between the two stars.

Ln0

 Double The initial value of of the z component of \(\hat{L}_N\)

χ1x

 Double The x component of star 1's angular momentum \(\vec{\chi}_1 \)

χ1y

 Double The y component of star 1's angular momentum \(\vec{\chi}_1 \)

χ1z

 Double The z component of star 1's angular momentum \(\vec{\chi}_1 \)

χ2x

 Double The x component of star 2's angular momentum \(\vec{\chi}_2 \)

χ2y

 Double The y component of star 2's angular momentum \(\vec{\chi}_2 \)

χ2z

 Double The z component of star 2's angular momentum \(\vec{\chi}_2 \)
Return Type Description
None N/A N/A
