Arrange 

CodeMethodetc.etc.

Parameters: none

Return: none

Technical description:

This method sets up things for solving the ATA matrix by first creating a string that contains the diagonal elements of the matrix. Then, this string is added to the ParamData() array. Finally, the normalization factors is set by dividing ATA(4,4) by the diagonal elements.

Theoretical description:

This method sets up the matrix that we will later invert to get uncertainty values. This matrix basically consists of partial derivatives of the wave with respect to the various parameters. The normalization factors ensure that the computer will be able to invert the matrix (if some values are much larger or smaller than others, the computer runs into trouble). These factors basically adjust the matrix so that the diagonal elements are identical, and they are "undone" after the inversion process.
