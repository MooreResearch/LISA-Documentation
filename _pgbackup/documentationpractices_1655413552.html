Documentation Practices

This page describes the process of handling documentation on the website.

Step 1: Download the current version of the website from the Box folder.

Step 2: Make the necessary edits to the pages you are tasked with updating.

Step 3: Create a new folder on your computer containing the pages you edited or created.

Step 4: Upload the new folder to Box, give it a description that explains the edits you made or the pages you created, and name it using the following convention: mm_dd_yyyy_(pages altered/created). (Example: 06_15_2022_documentation_theory.)

Step 5: A supervisor will look over your work and implement it into the main website folder as needed, thereby creating a new version of the website (old versions will be tracked in Box).
