GetdLNhatdot

Method (Function, i.e. spin, matrix, time evolution, etc.) Parent Class etc.

This method calculates and returns \( d\dot{\hat{L}}_N \)

Technical description:
The code first defines two vectors:  \( \vec{v}_1 = |\chi_1|(1+\delta)^2 \hat{\Omega}_1 \times\hat{\chi}_1 \) and  \( \vec{v}_2 = |\chi_2|(1-\delta)^2 \hat{\Omega}_2 \times \hat{\chi}_2 \). Then it calculates \(d \dot{\hat{L}}_N = - \frac{v}{4 \eta} (\vec{v}_1 + \vec{v}_2) \). Finally the code returns \(d \dot{\hat{L}}_N\). See the theororetical description for an expanation on how these equations were derived.
Parameter Type Description
v Double Approximately equal to \(v \over c \).

χ1hat

 Vector The unit vector that points in the direction of star 1's spin angular momentum.

χ1hat

 Vector The unit vector that points in the direction of star 2's spin angular momentum.

Ω1hat

 Vector The unit vector that points in the direction of \( \vec{\Omega}_1 \). See Equation 1.29b for the definition of \( \vec{\Omega}_1 \).

Ω2hat

 Vector The unit vector that points in the direction of \( \vec{\Omega}_2 \). See Equation 1.29b for the definition of \( \vec{\Omega}_2 \).
Return Type Description
dLNhatdot Vector The rate of change of the direction of the total angualar momentum of our system.
