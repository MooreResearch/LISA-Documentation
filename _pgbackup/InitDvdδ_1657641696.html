MethodSpin CaseEvolver

This method initializes \( \partial v \over \partial \delta \) by calculating and setting its first two values.

Technical description:
In order to initialize \( \partial v \over \partial \delta \) we must calculate both the past and current value of \( \partial v \over \partial \delta \). The code first calculates the past value of \( \partial v \over \partial \delta \) by creating two paralell simulations with differing values of \( \delta \) ; one simulation uses a value of \( \delta + \Delta \delta \) and the other uses \( \delta - \Delta \delta \). Currently the code uses a value of \( \Delta \delta = 10^{-6}\) The code then performs the cenetered difference between these two paralell simulations to obtain \( \partial \dot{v} \over \partial \delta \); the derivative of \( \partial v \over \partial \delta \) with respect to unitless time \(\tau_s = \frac{t_s}{M}\) in the source frame. The code then feeds this value to the EquationSolver class which contains analytic solitions for \( \partial v \over \partial \delta \) given \(v_0\) and \( \partial \dot{v} \over \partial \delta \). This becomes the past value of \( \partial v \over \partial \delta \), and the centered difference equation is used to obtain the present value of \( \partial v \over \partial \delta \).

Theoretical description:
This method provides the initial two values for \( \partial v \over \partial \delta \) that will be used to propegate the simulation forward in time. This method uses both the centered difference, and paralell simulations to determine the partial derivatives of v.
Usage locations:
This method is called only once in the constructor for the Evolver class.
