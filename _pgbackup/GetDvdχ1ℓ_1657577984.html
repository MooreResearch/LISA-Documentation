This method calculates and returns \(  \partial v \over \partial \chi_{1 \ell } \) given \(v\) and \( \partial \dot{v} \over \partial \chi_{1 \ell } \).

Technical description:
This method calculates and returns \( \partial v \over \partial \chi_{1 \ell } \) given \(v\) and \( \partial \dot{v} \over \partial \chi_{1 \ell } \). See the theoretical description for how a formula for \( \partial v \over \partial \chi_{1 \ell } \) was derived.
Parameter Type Description
v Double Approximately \( v \over c \)
dvdχ1ℓdot Double \( \partial \dot{v} \over \partial \chi_{1 \ell } \), the time derivative of \( \partial v \over \partial \chi_{1 \ell } \).
Return Type Description
dvdχ1ℓ Double \( \partial v \over \partial \chi_{1 \ell } \)
Theoretical description:
The formula for \( \partial v \over \partial \chi_{1 \ell } \) was obtained by rearranging Equation (5) of the DerivativeExpressions document to solve for \( \partial v \over \partial \chi_{1 \ell } \). 
Usage locations:
This method is called only once in the InitDvdχ1ℓ method of the Evolver class to intialize \( \partial v \over \partial \chi_{1 \ell } \). For all other calculations of \( \partial v \over \partial \chi_{1 \ell } \) the leapfrog method is used.
