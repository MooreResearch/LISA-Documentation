Evolver Constructor

MethodSpin CaseEvolver

Sets the intiail values \( d \tau \), \(v\),\(\delta\),\(\chi_{1 \ell}\), and \(\chi_{2 \ell}\) using parameters passed into the constuctor. Also initializes the first two values of \(v\), \( \partial v \over \partial \delta \), \( \partial v \over \partial \chi_{1 \ell} \), and \( \partial v \over \partial \chi_{2 \ell} \). 

Technical description:
This constructor uses parameters passed into it to set initial values for \( d \tau \), \(v\),\(\delta\),\(\chi_{1 \ell}\), and \(\chi_{2 \ell}\). The code then calls InitV, InitDvdδ, InitDvdχ1ℓ, and InitDvdχ2ℓ to set up the first two values of \(v\), \( \partial v \over \partial \delta \), \( \partial v \over \partial \chi_{1 \ell} \), and \( \partial v \over \partial \chi_{2 \ell} \) respectively.
Parameter Type Description
mydτ0 Double The initial timestep to be used in the centered difference equation to propegate each value.

myv0

 Double The initial value of v, equal to \((2\pi M f_0)^{1/3} \)

myδ

 Double The intial value of  \(\delta = \) \( m_1 - m_2 \over m_1 + m_2 \)

myχ1ℓ

 Double The initial value of \( \chi_{1 \ell} \), which is passed in by the user

myχ2ℓ

 Double The initial value of \( \chi_{2 \ell} \), which is passed in by the user
